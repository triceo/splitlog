About splitlog: watching your logs

Tool to monitor log files from your Java applications as they fill.

Project Modules

This project has declared the following modules:

Name Description
splitlog-api: the public APIs These are the APIs of Splitlog where we are ready to commit to increased stability.
splitlog-core: the client library These are the core parts of splitlog. This is where the implementations of the public APIs live. Classes from this package come with no API stability guarantees.
